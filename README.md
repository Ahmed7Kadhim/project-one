# Wear Calculator (Galaxy Watch / Wear OS)

Simple offline Wear OS calculator app built with Kotlin.

## Features
- Basic operations: `+`, `-`, `×`, `÷`, `%`
- Decimal division output with precise BigDecimal math
- Clear button and chained operations
- Dark UI with large touch-friendly buttons
- Separate expression and result areas
- Optional UX extras included:
  - Haptic feedback on button press
  - Last expression/result persistence
  - Rotary bezel scroll support for content navigation

## Build
Open in Android Studio and sync Gradle, or run:

```bash
./gradlew test
./gradlew assembleDebug
```
